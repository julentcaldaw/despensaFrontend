# Personas

## 1. Household Organizer
Profile: Manages weekly groceries for a family or shared home.

Needs:
- Know what is available before shopping.
- Avoid expired products and duplicate purchases.
- Build shopping lists quickly.

Pain points:
- Pantry visibility is fragmented.
- Meal planning takes too much time.

## 2. Practical Solo User
Profile: Lives alone, wants simple routines with minimal app friction.

Needs:
- Fast item updates.
- Easy recipe ideas using current ingredients.
- Lightweight reminders through list suggestions.

Pain points:
- Forgets what is at home.
- Buys the same item twice.

## 3. Contributor User
Profile: Shared-home member who collaborates on pantry and shopping tasks.

Needs:
- Update inventory without breaking data consistency.
- Add useful list items for others.

Pain points:
- Unclear ownership of changes.
- Inconsistent naming of pantry items.

## Role mapping
- `user`: Uses pantry, recipes, shopping list, and profile.
- `contributor`: Same as `user` plus collaboration-focused updates in shared flows.
- `admin`: Moderates content and manages operational settings.
