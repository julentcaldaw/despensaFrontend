# Project Context For Agents

## Product summary
Despensa is a React + TypeScript web app for pantry control, shopping list support, and recipe discovery from available ingredients.

## Current phase
Initial production release planning and implementation.

## Core routes
- `/register`
- `/login`
- `/pantry`
- `/recipes`
- `/shopping-list`
- `/profile`

## Roles
- `user`
- `contributor`
- `admin`

## Assumptions
- Backend API exists but may evolve.
- Recipes are sourced from internal data.
- Shopping list supports manual input plus automatic suggestions.

## Agent objective
Deliver complete, working changes with minimal ambiguity and clear documentation updates.
