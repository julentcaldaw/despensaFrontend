# Definition Of Done

## Required checks
- Requested functionality is fully implemented.
- Behavior works correctly in target routes.
- Errors and empty states are handled.
- Role-based behavior is respected.
- Relevant docs are updated.

## Code quality checks
- No TypeScript or lint errors introduced.
- Naming and folder conventions are respected.
- Server-state updates and invalidations are correct.

## Validation checks
- At least one manual validation pass of the affected flow.
- Any known limitations are explicitly documented.
